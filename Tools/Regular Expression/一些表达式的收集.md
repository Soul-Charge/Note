# 一些表达式的收集

## 匹配双引号内的字符

匹配两个双引号之间没有双引号的字符串  

```regex
"[^"]*"
```

## 匹配12小时制的时间

```regex
(1[012]|[1-9]):([0-5][0-9]) (am|pm)
```

## 匹配24小时制时间

小时的十位不可省略：

```regex
(([01][0-9])|(2[0123])):([0-5]|[0-9])
```

小时的十位可省略

```regex
(0?[0-9]|1[0-9]|2[0-3]):([0-5][0-9])
```

```regex
([01]?[0-9]|2[0-3]):([0-5][0-9])
```
