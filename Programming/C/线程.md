# 线程

## POSIX线程

使用pthread库，需要包含的头文件是`pthread.h`


### 线程函数

```c
void *fun(void) // 新线程执行的函数
{
    // code...
    return NULL;
}
```

> 线程执行的函数应返回空指针(void*)

### 创建线程

```c
pthread_t t; // 保存线程信息的结构
if (pthread_create(&t, NULL, fun, NULL) == -1)
{
    // 返回-1即无法创建线程
}
```

> fun为函数名(函数指针)，见上面(###线程函数)

### 回收线程

```c
void* result;
if (pthread_join(t, &result) == -1)
{
    // 返回-1即无法回收线程
}
// t是pthread_t类型变量,见上面(###创建线程)
```
